# SR Flip-Flop

## Overview
An SR (Set-Reset) Flip-Flop is a basic sequential logic circuit used to store 1-bit of data.

## Port Description

| Port Name | Direction | Width  | Description |
| --------- | --------- | ------ | ----------- |
| `clk`     | Input     | 1-bit  | Clock signal |
| `s`       | Input     | 1-bit  | Set input |
| `r`       | Input     | 1-bit  | Reset input |
| `q`       | Output    | 1-bit  | Data output |
