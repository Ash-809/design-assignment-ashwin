# 4x2 Decoder

## Overview
A 4x2 decoder is a combinational logic circuit that takes 2 binary inputs and activates one of the 4 unique outputs based on the input combination.

## Port Description

| Port Name | Direction | Width  | Description |
| --------- | --------- | ------ | ----------- |
| `in`      | Input     | 2-bit  | Binary input |
| `out`     | Output    | 4-bit  | Decoded output |
